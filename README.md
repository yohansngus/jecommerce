What it does:
An online shopping app where users can browse products, add items to a cart, and manage purchases.

Problem it solves:
Provides a simple and user-friendly way to shop online without confusion or clutter.

Tech used:
React / Next.js
JavaScript
CSS / Tailwind
Vercel (deployment)

Features:
Product listing
Add / remove items from cart
Responsive design
Deployed live
